# 🎬 Streamlit Movie Recommendation System

A simple **Content-Based Movie Recommendation System** built using **Python, Streamlit, Pandas, and Scikit-learn**.

This app recommends movies based on the **genre similarity** of the movie you select.

---

## 🚀 Features

- Content-Based Filtering
- Cosine Similarity
- Streamlit Web Interface
- Clean project structure (model, utils, dataset)
- Easy to extend with real datasets

---

## 🧠 Concept Used

- CountVectorizer (Text Vectorization)
- Cosine Similarity (Similarity Measurement)
- Streamlit (Frontend UI)
